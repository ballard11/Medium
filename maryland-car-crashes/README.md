# Maryland Car Crashes

**Status:** Draft

## Topic
Time-series analysis of Maryland vehicle crashes, including COVID-19 impact and fatal crash trends across top counties.

## Data Sources
- [Maryland Statewide Vehicle Crashes](https://opendata.maryland.gov/Public-Safety/Maryland-Statewide-Vehicle-Crashes/65du-s3qu) — OpenData Maryland

## Notes
The full dataset (`Maryland_Statewide_Vehicle_Crashes.csv`) must be downloaded from the link above and placed in the `data/` folder. The included `small_Maryland.csv` is a filtered subset.
